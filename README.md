# Undangan Digital Peneguhan Sidi

Website undangan digital yang elegan untuk acara peneguhan sidi Ecca Febriana & Cristin Purba.

## 📋 Cara Menggunakan

### 1. Persiapan Gambar
Simpan gambar undangan Anda dengan nama **`invitation-cover.jpg`** di folder yang sama dengan file `index.html`.

**PENTING:** Nama file harus **PERSIS** `invitation-cover.jpg` (huruf kecil semua).

### 2. Struktur File
Pastikan struktur folder Anda seperti ini:
```
📁 folder-website/
├── index.html
├── style.css
├── script.js
├── invitation-cover.jpg  ← Gambar undangan Anda
└── README.md
```

### 3. Membuka Website
1. Buka file `index.html` di browser (Chrome, Firefox, Edge, dll)
2. Gambar undangan akan muncul sebagai background
3. Tombol "BUKA UNDANGAN" akan muncul di bagian bawah dengan animasi
4. Klik tombol untuk melihat isi undangan lengkap

## ✨ Fitur Website

### Halaman Pembuka
- Background menggunakan gambar undangan Anda
- Tombol "Buka Undangan" dengan animasi pulse
- Efek konfetti saat tombol diklik
- Transisi smooth ke halaman utama

### Halaman Utama
- **Section Home:** Sambutan dan ayat Alkitab
- **Section Acara:** Detail lengkap acara peneguhan sidi
- **Section Galeri:** Tempat untuk foto-foto (dapat dikustomisasi)
- **Section Lokasi:** Alamat lengkap dan link Google Maps
- **Navigasi:** Smooth scroll antar section
- **Responsive:** Tampil sempurna di desktop, tablet, dan mobile

## 🎨 Kustomisasi

### Mengganti Gambar
Jika ingin mengganti gambar, cukup replace file `invitation-cover.jpg` dengan gambar baru (pastikan nama file tetap sama).

### Menambahkan Musik Background
1. Siapkan file musik (format MP3)
2. Simpan dengan nama `background-music.mp3` di folder yang sama
3. Musik akan otomatis diputar saat undangan dibuka

### Menambahkan Foto ke Galeri
Edit file `index.html` di section gallery dan ganti placeholder dengan foto Anda.

## 🔧 Troubleshooting

### Gambar Tidak Muncul?
- Pastikan nama file adalah `invitation-cover.jpg` (huruf kecil)
- Pastikan file gambar ada di folder yang sama dengan `index.html`
- Coba refresh browser (tekan F5 atau Ctrl+R)
- Coba buka di browser berbeda

### Tombol Tidak Berfungsi?
- Pastikan file `script.js` ada di folder yang sama
- Buka Console di browser (F12) untuk melihat error
- Pastikan JavaScript tidak diblokir oleh browser
